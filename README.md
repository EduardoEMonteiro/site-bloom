Bloom Hair – Website Profissional para Salão de Beleza

Website desenvolvido com foco em experiência do usuário, performance e presença digital para salões. O projeto demonstra domínio em HTML5 semântico, CSS3 moderno com variáveis e media queries, JavaScript para interações e otimizações de SEO técnico avançado.

O site é totalmente estático, otimizado para carregamento rápido, com integrações de terceiros (Google Maps, Spotify, Instagram) e totalmente responsivo.

Visão Geral do Projeto

O objetivo deste projeto é criar uma presença digital premium para o salão Bloom Hair, comandado pela profissional Tamires Sousa. O site foi pensado para converter visitantes em clientes através de uma experiência visual imersiva, informações claras e calls-to-action estratégicos.

Principais objetivos:

· Performance superior a 95 no Google PageSpeed
· Design que transmite elegância e profissionalismo
· Experiência mobile-first fluida e intuitiva
· SEO local otimizado para "salão São João Del Rei"
· Integração perfeita com redes sociais e WhatsApp
· Acessibilidade e semântica HTML de qualidade

Destaques Técnicos

· HTML5 semântico com estrutura lógica e acessível
· CSS3 moderno com variáveis CSS, Flexbox, Grid e media queries avançadas
· JavaScript vanilla para interações suaves (reveal animations)
· Otimização de performance com lazy loading, preconnect e font-display
· WebP e otimização de imagens para redução de peso
· SEO técnico completo com Schema.org, Open Graph, meta tags
· Integrações externas (Google Maps, Spotify, Instagram via Elfsight)
· Design system consistente com paleta de cores e tipografia profissional

Destaques de UX/UI

· Hierarquia visual clara com foco na jornada do usuário
· Paleta de cores exclusiva baseada em verde escuro (#012220) e dourado (#fff8af)
· Tipografia profissional (Montserrat para legibilidade e modernidade)
· Componentes responsivos com transições suaves
· Microinterações em botões e elementos interativos
· CTA principal destacado com animação "pulse" para atenção
· Layout balanceado com uso estratégico de espaços em branco
· Navegação simplificada (single page com scroll suave)

Sistema de Design

Paleta de Cores

· Verde escuro (#012220) - sofisticação, natureza, estabilidade
· Dourado (#fff8af) - luxo, qualidade, atenção aos detalhes
· Preto (#000000) - contraste e profundidade
· Branco/off-white (#e8e8e8) - legibilidade e limpeza

Tipografia

· Montserrat (300-600) para todos os textos - moderna, legível, versátil

Microinterações

· Hover effects em botões sociais e links
· Animação "pulse" no CTA principal
· Reveal animations no scroll
· Transições suaves em todos os estados interativos

Componentes

· Header fixo com logo centralizada
· Cards com sombras e bordas arredondadas
· Grids responsivos (mobile-first)
· Formulários com validação nativa

Stack Tecnológica

Frontend:

· HTML5 semântico
· CSS3 (variáveis, Flexbox, Grid, media queries)
· JavaScript ES6+ (vanilla)

Design:

· Google Fonts (Montserrat)
· Font Awesome 6 (ícones)
· Sistema de design próprio

Integrações:

· Google Maps API
· Spotify Embed
· Instagram via Elfsight
· WhatsApp Business
· Schema.org structured data

Performance:

· Critical CSS inline
· Lazy loading de imagens e iframes
· Preconnect para recursos externos
· Font optimization (font-display: swap)
